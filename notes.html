HTML SEMANTIC TAGS

Section
-The <section> element defines a section in a document.
Article
-The <article> element specifies independent, self-contained.
Header
-The <header> element represents a container for introductory contentor a set if navigational links.
Footer
-The <footer> element defines a footer for a document or section.

HTML BASIC TAGS

Heading
-The <h1> element defines most important headings.
-The <h6> element defines least important headings.
Paragraph
-The <p> element defines paragraph.
